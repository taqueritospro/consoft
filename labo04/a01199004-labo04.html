Preguntas

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java se ejecuta en una JVM además de que es estático y robusto, mientras que JavaScript se ejecuta en navegadores, es dinámico y flexible

¿Qué métodos tiene el objeto Date? (Menciona al menos 5)

getFullYear(), getMonth(), getDate(), getHours(), getMinutes().

¿Qué métodos tienen los arreglos? (Menciona al menos 5)

at(), concat(), find(), join(), forEach().

¿Cómo se declara una variable con alcance local dentro de una función?

Usando let o const dentro de la función.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Pueden causar errores si varias funciones la usan, además es más dificil el manejo de estos scripts.
